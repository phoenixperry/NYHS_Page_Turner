Packagestarling.extensions.brinkbit.fullscreenscreenextension
Classpublic class FullScreenExtension
InheritanceFullScreenExtension Object

A static class to simplify multi-resolution support in Starling. When creating a new instance of Starling, call ScreenExtension.createStarling() instead of new Starling().

See http://spacebarup.com for tutorials and more information.

createStarling()method 
public static function createStarling(rootClass:Class, flashStage:Stage, width:int, height:int, horizontalGravity:String = center, verticalGravity:String = center, manuallySize:Boolean = false):Starling

Creates a new instance of Starling and initializes the stage.

Parameters

rootClass:Class — A subclass of a Starling display object. It will be created as soon as initialization is finished and will become the first child of the Starling stage.
 
flashStage:Stage — The Flash (2D) stage.
 
width:int — The width of the ScreenExtension stage.
 
height:int — The height of the ScreenExtension stage.
 
horizontalGravity:String (default = center) — The horizontal alignment of the stage relative to the screen. Accepts HAlign constants.
 
verticalGravity:String (default = center) — The vertical alignment of the stage relative to the screen. Accepts VAlign constants.
 
manuallySize:Boolean (default = false) — Turns off autoResizing. If true, the screen and stage will not be positioned or sized until you manually call resize();

Returns
Starling
